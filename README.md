# Daimon

**Daimon** (Greek: δαίμων — guiding spirit) — AGNOS agent orchestrator.

The core runtime for the AGNOS ecosystem: agent lifecycle, HTTP API (port 8090), process supervision, IPC, task scheduling, multi-node federation, edge fleet management, memory/vector/RAG stores, MCP tool dispatch (151 tools), and screen capture.

## Status

Extraction complete. HTTP API, tests, and benchmarks in place. Hardening in progress toward v1.0.
